
## 1) 제품 개요

### 핵심 컨셉
* 지도 기반 후기 공유 커뮤니티
- 사용자는 **주제별 모임(=커뮤니티/게시판)** 을 만들고, 누구나 가입 가능(공개 기본).    
- 모임 가입자는 **지도 좌표(위치)에 핀 형태로 글 / 평점(정보) / 후기** 를 작성.
- 글은 **작성자만 수정/삭제** 가능.
- 모임 가입자만 해당 모임의 글 열람 가능.
- 사용자는 가입한 모임의 글을 **리스트(근거리 글)** 와 **지도(핀)** 두 방식으로 탐색.
- **특정 사용자의 글 차단** 가능.

### 주요 용어(엔티티)
- **User**: 사용자
- **Group(모임)**: 주제별 커뮤니티
- **Membership(가입)**: 유저-모임 관계
- **Post(핀 글)**: 특정 좌표에 작성된 글
- **Block(차단)**: 유저가 특정 유저를 차단

## 2) 권한/정책(필수 규칙)

### 권한 매트릭스
- 모임 생성: 로그인 사용자
- 모임 가입: 공개모임(누구나), 비공개모임(초대링크)
- 글 작성/열람:
    - **작성**: 모임 가입자만
    - **열람**: **모임 가입자만**
- 글 수정/삭제: **작성자만**
- 차단: 사용자 개인 설정(차단한 사용자의 글은 **내 화면에서 숨김**)
* 신고: 광고성 게시글 및 정책에 어긋난 게시글 신고

### 차단 정책(추천 세부 규칙)
- A가 B를 차단하면:
    - A는 B의 글/댓글/프로필 일부(닉네임 정도) 노출 안함
    - A의 화면에서는 B 콘텐츠가 전부 필터링
    - B가 A 콘텐츠를 보지 못하게 “상호 차단”

## 3) 기능명세 (우선순위 포함)

### P0 (MVP: 반드시)
#### 3.1 인증/계정
- 소셜 로그인(Apple/Google/Kakao 등) 또는 이메일
- 닉네임, 프로필 이미지
- 위치 권한 동의(앱 사용성에 중요)

#### 3.2 모임(커뮤니티)
- 모임 생성
    - 필드: 이름, 주제/카테고리, 소개, 대표 이미지
    - 공개 여부: **공개(기본)** (비공개는 추후 개발)
*  모임 관리자
	* 모임을 생성한 사람은 관리자가 된다
	* 그룹 정보 수정(이름/소개/이미지/카테고리)
	- 그룹 폐쇄(신규 글 작성 불가/ 단 기존 가입자들은 이전 작성된 글들은 열람은 가능)     
- 모임 목록/검색
    - 카테고리 필터(사우나/맛집/카페/붕어빵/야경/숨은 관광지 등)
    - 키워드 검색(모임명)
        
- 모임 가입/탈퇴
    - 가입 즉시 멤버십 생성
    - 탈퇴 시 해당 모임의 게시글 볼 수 없음
        
- 모임 상세
    - 소개, 멤버 수, 최근 글 수, “가입/탈퇴” 버튼
    
#### 3.3 핀 글(좌표 글)
- 글 작성(모임 내부)
    - 위치 좌표(현재 위치/지도에서 선택)
    - 평점, 본문, 사진(선택), 태그(선택)
    - 작성 시점/수정 시점 저장
        
- 글 열람(모임 가입자)
    - 리스트(근거리/최신) + 지도 핀에서 열람
        
- 글 수정/삭제(작성자만)
    - 서버에서 작성자 검증
        
- 근거리 탐색
    - 반경 선택: 500m / 1km / 3km / 5km 등
    - 정렬: 거리순 / 최신순

#### 3.4 지도/리스트 뷰
- 모임 내 화면에서 **“리스트 ↔ 지도 토글”**
- 지도에서 핀 클러스터링(핀 많아질 때 필수)
- 핀 탭 → 미리보기 카드 → 상세 화면 진입

#### 3.5 차단
- 게시글/프로필에서 “이 사용자 차단”
- 차단 목록 관리(설정)
- 차단한 사용자의 글은 조회 API에서 제외(또는 클라이언트 필터링 + 서버에서도 2차 필터링 권장)
-  **기본: 단방향 숨김(내 화면에서만 숨김)**
- 옵션으로 “상호 차단” 제공(설정에서 토글)

### 추가 명세 필요 사항
- 댓글/대댓글, 좋아요(혹은 “도움돼요”)
- 신고(게시글/사용자) + 운영자 처리 플로우
- 모임별 공지/고정글
- 알림(Push): 내가 가입한 모임에 새 글, 내 글에 댓글 등
- 비공개/승인제 모임
- 사용자 레벨/배지(기여도) 또는 신뢰지표(스팸 억제)
- 장소(POI) 연동: Naver map으로 장소 선택
    

## 7) UI/UX 정보구조(IA) 제안

### 하단 탭(모바일 기준)
1. **근처(Nearby)**: 내가 가입한 모임들의 “내 주변 글” 피드
2. **지도(Map)**: 지도에서 모임/글 탐색(토글로 그룹 필터)
3. **모임(Groups)**: 모임 탐색/가입/내 모임
4. **더보기(Profile)**: 프로필 설정/차단 목록

## 8) 화면별 UX 플로우(구체)

### 8.1 온보딩

- 1. 로그인 → 2) 닉네임 설정 → 3) 위치 권한 요청
- 위치 권한 거부 시:
    - 가입 불가

### 8.2 모임 탐색/가입
- 모임 리스트 카드:
    - 커버/이름/카테고리/멤버 수/최근 글 수
        
- 상세 화면:
    - 상단 커버 + 소개
    - “가입하기” CTA
    - 가입 후: “글 보기 / 지도 보기” 진입
        

### 8.3 모임 내부(핵심)
- 상단: 모임 이름 + 검색 아이콘 + 필터
- 가운데: **리스트/지도 토글**
- 하단: “+ 핀 글 작성” 플로팅 버튼

**리스트 모드**
- 정렬: 거리순/최신순
- 반경 선택 드롭다운
- 글 카드: 제목/대표사진/거리/작성자/작성시간/간단 본문
    

**지도 모드**
- 핀 클러스터 → 탭 시 확대 또는 리스트로 펼치기
- 핀 탭 → 바텀시트 미리보기(제목/사진/거리) → 상세
- 평균점수 및 가입한 모임의 후기 조회 가능
    
### 8.4 글 작성 UX(실사용성 중요)
- Step1: 위치 선택
    - 현재 위치로 자동 세팅 + “지도에서 선택” 버튼
    - (추천) 위치 미세조정 드래그
- Step2: 내용 작성
    - 평점/본문/사진
- Step3: 공개 범위 
	- 선택한 모임만 공개
- 완료 후:
    - “등록됨” 피드백
        

### 8.5 글 상세
- 평점 / 작성일 / 본문
- 신고/차단/공유(정책에 따라)
- 작성자만 “수정/삭제” 노출
    

### 8.6 차단 UX
- 글 상세 ⋮ 메뉴 → “이 사용자 차단”
- 차단 시 토스트: “이 사용자의 글이 숨겨집니다. 설정에서 해제 가능”
- "상대방에게도 내 글 안보여주기" 선택 가능
- 프로필 > 설정 > 차단 목록(해제 버튼)
    
## 10) MVP 개발 범위
- 로그인/프로필
- 모임 생성/검색/가입/탈퇴
- 모임 내 글 CRUD(작성자만 수정/삭제)
- 근거리 리스트 + 지도 핀(클러스터)
- 차단(내 화면에서 숨김 + 서버 필터)
- 기본 운영 로그/레이트리밋
